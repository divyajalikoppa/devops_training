Remote Login Using Secure Shell (SSH)

Step 1: Login to CentOS

Login to your CentOS machine as the root user with your password.

Step 2: Install SSH

Run the following command to install SSH on your server:

sudo yum -y install openssh-server openssh-clients

Step 3: Configure SSH

Open the SSH configuration file:

sudo vi /etc/ssh/sshd_config

Modify the following lines:


    Remove "#" from Port 22
    Remove "#" from PermitRootLogin yes
    Remove "#" from PasswordAuthentication yes
    Remove "#" from PubkeyAuthentication yes

Step 4: Restart SSH

Restart the SSH server with the following command:

sudo systemctl restart sshd

Step 5: VirtualBox Configuration

Open your VirtualBox and make the changes. Go to network settings and in adapter 1 attach it to 'host only adapter'.

Step 6: Verify CentOS IP

On your CentOS server, open a terminal and run the command:

ip a

Verify that an IP address is generated.

Step 7: SSH from Windows

Open your Windows Command Prompt and run the following SSH command to connect to your CentOS server:

ssh username@yourip

Replace "username" with your CentOS username and "yourip" with the CentOS server's IP address. Verify that you can access CentOS using SSH.
