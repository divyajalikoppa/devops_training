FileZilla File Transfer

Step 1: Download FileZilla Client

Open your web browser and navigate to the FileZilla download page:

FileZilla Download Page

Step 2: Download and Install

Download the FileZilla Client and install the application on your Windows machine.

Step 3: Open FileZilla

Launch the FileZilla application.

Step 4: Configure New Site

In FileZilla, navigate to File → Site Manager → New Site.

Step 5: Provide Connection Details

Provide the following details:

  • Host: Your CentOS machine's IP address
  • Port: 22 (default for SSH)

Click "Connect" to proceed.

Step 6: Successful Connection

If the connection is successful, you will see a window listing all the files and folders on your CentOS machine.

Step 7: Transfer Files

Now, you can transfer files from your Windows machine to your Linux CentOS machine. Simply drag and drop the files from your local directory to the remote directory on the CentOS machine.
